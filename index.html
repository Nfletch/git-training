Introduction to Git (and GitHub)

Forking a repository on GitHub

  1. Create an account on github.com
  2. Fork this repository: https://github.com/jvalleroy/reveal.js

Creating a local clone

  1. git clone https://github.com/username/reveal.js.git
  2. cd reveal.js

Staging a change

  1. Edit index.html in text editor
  2. git add index.html

Committing a change

  1. git commit -m "Change slide text"

Viewing status

  1. git status

Viewing the log

  1. git log

Viewing differences

  1. git diff origin/git-training

Pushing a change

  1. git push origin/git-training

Pulling changes

  1. git pull origin/git-training

Creating a branch

  1. git checkout -b fix-spelling-errors

Pushing a branch

  1. git push origin/fix-spelling-errors

Creating a pull request on GitHub

Merging a pull request on GitHub

Adding a remote repository

  1. git remote add upstream https://github.com/hakimel/reveal.js.git

Fetching changes from a remote

  1. git fetch upstream

Checking out a remote branch

  1. git checkout -b upstream-master upstream/master

Rebasing a branch

  1. git rebase upstream-master

Merging a branch

  1. git merge upstream-master

Deleting a branch

  1. git branch -D upstream-master
